# Bilibili弹幕数据分析系统 - 优化文档

> 本文档记录了对原始项目的优化和新增功能。

## 一、项目概述

**目标视频**: BV1jEAaz3E6K (一个视频搞懂OpenClaw！)
**原始功能**: 爬虫 + NLP分词/词频/情感/LDA + HBase存储 + ECharts可视化

---

## 二、优化内容

### 2.1 NLP模块增强

| 脚本文件 | 功能描述 | 输出文件 |
|---------|---------|---------|
| `segmentation.py` | **pkuseg分词升级** - 基于CRF的条件随机场分词，精度高于jieba | `wordfreq_pkuseg.json` |
| `ner_recognition.py` | **NER实体识别** - 基于词性标注识别人名/地名/机构名/专有名词 | `ner_entities.json` |
| `sentiment_enhanced.py` | **情感分析增强** - 规则+词典+否定处理+表情符号+标点分析 | `sentiment_enhanced.json` |
| `danmaku_time_distribution.py` | 分析弹幕在视频各时间段的分布密度 | `danmaku_time_distribution.json` |
| `user_behavior_analysis.py` | 基于用户ID分析发送弹幕次数、活跃时段、情感倾向 | `user_behavior.json` |
| `sentiment_trend.py` | 分析视频播放过程中情感变化趋势 | `sentiment_trend.json` |
| `danmaku_classifier.py` | 基于规则识别祝福类、玩梗类、刷屏类、提问类等弹幕 | `danmaku_classified.json` |
| `keyword_extraction.py` | 使用TF-IDF和TextRank算法抽取关键词 | `keywords.json` |
| `word_cooccurrence.py` | 分析词语之间的共现关系，构建词语网络 | `word_cooccurrence.json` |

#### 2.1.1 分词升级 (pkuseg)

```bash
python segmentation.py
```

**pkuseg优势**:
- 基于条件随机场(CRF)的中文分词
- 在多个公开数据集上分词准确率高于jieba
- 支持词性标注

```python
# 使用示例
from segmentation import Segmenter
seg = Segmenter()
words = list(seg.cut_with_pos('一个人工智能视频'))
# [('一', 'n'), ('个人', 'n'), ('工', 'n'), ('智', 'n'), ('能', 'n'), ('视频', 'n')]
```

#### 2.1.2 实体识别 (NER)

```bash
python ner_recognition.py
```

**识别实体类型**:

| 实体类型 | 词性标签 | 示例 |
|---------|---------|------|
| 人名 | nr | 蔡徐坤、老师、同学 |
| 地名 | ns | 北京、上海、东京 |
| 机构名 | nt | 谷歌、阿里、腾讯 |
| 专有名词 | nz | AI、OpenClaw |
| 产品词 | 自定义 | iPhone、ChatGPT |
| 技术词 | 自定义 | AI、机器学习 |

**输出字段**:
```json
{
  "total_danmaku": 3514,
  "danmaku_with_entities": 1234,
  "entity_ratio": 35.1,
  "entity_stats": {
    "person": {"count": 234, "unique_count": 45, "top_entities": {...}},
    "location": {"count": 123, "unique_count": 28, "top_entities": {...}},
    "tech": {"count": 567, "unique_count": 89, "top_entities": {...}}
  }
}
```

#### 2.1.3 情感分析增强

```bash
python sentiment_enhanced.py
```

**改进点**:

| 特性 | 原有方案 | 增强方案 |
|------|---------|---------|
| 否定词处理 | 无 | 完整否定词表 |
| 程度副词 | 无 | 权重放大/缩小 |
| 表情符号 | 无 | 正面/负面表情库 |
| 标点分析 | 无 | 感叹号增强 |
| 转折句式 | 无 | 检测试验 |

**示例结果**:
```
"这个视频太棒了，学到很多！" -> positive (score: 1.0)
"垃圾视频，浪费时间" -> negative (score: -1.0)
"一般般吧，没什么特别的" -> neutral (score: 0)
"哈哈哈笑死我了" -> positive (score: 1.0)
```

#### 2.1.1 弹幕时间分布分析

```bash
python danmaku_time_distribution.py
```

**分析维度**:
- 按30秒/60秒分桶统计弹幕数量
- 识别高密度时段（弹幕最密集的时间点）
- 按时长分组统计（0-5min, 5-10min, 10-15min等）
- 生成直方图数据用于前端可视化

**输出字段**:
```json
{
  "total_danmaku": 3514,
  "bucket_size_seconds": 30,
  "total_buckets": 120,
  "avg_per_bucket": 29.3,
  "high_density_buckets": [...],
  "time_segments": {
    "0-5min": 892,
    "5-10min": 1456,
    "10-15min": 856,
    "15-20min": 310
  },
  "peak_bucket": {"time_seconds": 180, "count": 87}
}
```

#### 2.1.5 关键词抽取

```bash
python keyword_extraction.py
```

**算法**:
- **TF-IDF**: 词频-逆文档频率
- **TextRank**: 基于词语共现网络的PageRank变体
- **综合得分**: 归一化后两种方法各占50%权重

**输出字段**:
```json
{
  "tfidf_keywords": [
    {"word": "AI", "score": 0.0234},
    {"word": "权限", "score": 0.0198}
  ],
  "textrank_keywords": [
    {"word": "AI", "score": 1.234},
    {"word": "模型", "score": 0.987}
  ],
  "combined_keywords": [
    {
      "word": "AI",
      "combined_score": 0.854,
      "tfidf_score": 0.0234,
      "textrank_score": 1.234
    }
  ]
}
```

```bash
python word_cooccurrence.py
```

**分析维度**:

- 按词语类别（positive/negative/neutral）分组

**输出字段**:

```json
  "top_cooccurrence_pairs": [
    {"word1": "AI", "word2": "模型", "count": 45}
```

## 七、注意事项

1. **数据依赖**：部分新增脚本依赖 `cleaned_danmaku.json` 和 `sentiment.json`，需先运行清洗和情感分析脚本
2. **跨域访问**：动态加载JSON需通过HTTP服务器访问，避免浏览器跨域限制
3. **运行顺序**：
   ```bash
   # 1. 原有NLP流程
   python nlp_process.py           # 词频统计
   python sentiment_lexicon.py      # 情感分析

   # 2. 增强NLP流程
   python segmentation.py           # pkuseg分词
   python sentiment_enhanced.py     # 增强情感分析
   python ner_recognition.py       # 实体识别

   # 3. 新增分析
   python danmaku_time_distribution.py
   python user_behavior_analysis.py
   python sentiment_trend.py
   python danmaku_classifier.py
   python keyword_extraction.py
   python word_cooccurrence.py
   ```
